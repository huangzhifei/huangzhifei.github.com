关于 Apple Developer Programs 的申请基本流程，请参考以下三个步骤：

第一步：APPLE ID

在申请开发者账号之前，您需要一个有效的 Apple ID。如果您尚未申请 Apple ID，请至以下网站点选 Create an Apple ID (创建 Apple ID)提交申请：

第二步：D-U-N-S NUMBER（如您准备申请的是个人版本的计划，可跳过此步骤）

公司版和企业版的 Apple 开发者帐号申请需要您提交 D-U-N-S NUMBER。您可使用以下链接免费获取 D-U-N-S NUMBER，申请的表格需要您用英文或拼音填写：

https://developer.apple.com/program/enroll/dunsLookupForm.action

D-U-N-S NUMBER 常见问题：

https://developer.apple.com/cn/support/ios/D-U-N-S.html

＊D-U-N-S NUMBER 的申请时间约需要七个工作日。

第三步：加入 APPLE DEVELOPER PROGRAM

Apple Developer Programs 之资料如下：

一年¥688元 - 公司

- 此计划是对于希望开发 Apple 的应用程序的公司而设。

- 参加此计划其公司名称将作为卖方在 App Store 上进行应用销售。
